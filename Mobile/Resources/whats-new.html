Features

  • Push notifications of highlights and private messages when using a supported IRC bouncer.
  • Colloquy Bouncer integration makes staying connected with push notifications through your Mac simple.
  • Server room list browser and the /list command.
  • Post to Twitter using the /tweet command.
  • Save links to Instapaper from the built-in browser.
  • Broadcast the currently playing song using the /music command.

New

  • Audio notifications for highlights and private messages.
  • Sleek black interface when using a dark transcript style.
  • Browse built-in help topics using the /help command.
  • Choose from a built-in list of 25 popular networks.
  • Find chat rooms with SearchIRC.com using the /search command.
  • Setting to disable history when reconnecting.
  • Setting to disable graphical emoticons.
  • Setting to show full room names.
  • Settings to disable vibration.
  • Initial support for accessibility on iPhone 3GS.
  • Optimized for iPhone 3GS.
  • Localized in German.

Changes

  • Improved responsiveness when receiving a backlog of chat messages.
  • Better memory management for longer chat sessions.
  • Stays connected in more cases while the device is locked.
  • Identify properly services on more networks.
  • More screen space for the chat in landscape mode.
  • Notices are shown by default (with a setting to change the behavior).
  • Fixes top reported crashes.
  • Fixes for iPhone OS 3.0.
